# House Price Prediction

## 📌 Project Overview

This project predicts house prices using Machine Learning based on various property features such as area, number of bedrooms, bathrooms, parking spaces, furnishing status, and other amenities. The objective is to understand the factors affecting house prices and build predictive models for real estate price estimation.

---

## 🎯 Objectives

- Explore and understand the housing dataset.
- Clean and preprocess the data.
- Perform Exploratory Data Analysis (EDA).
- Build and compare Machine Learning regression models.
- Evaluate model performance using regression metrics.
- Visualize relationships between house features and prices.
- Provide insights for real estate decision-making.

---

## 📂 Dataset

- **Dataset:** Housing.csv
- **Records:** 545 Houses
- **Features:** 13 Input Features + 1 Target (Price)

---

## 🛠️ Technologies Used

- Python
- Google Colab / Jupyter Notebook
- Pandas
- NumPy
- Matplotlib
- Seaborn
- Scikit-learn

---

## 🤖 Machine Learning Models

- Linear Regression
- Random Forest Regressor

---

## 📊 Model Evaluation

The models were evaluated using:

- Mean Absolute Error (MAE)
- Root Mean Squared Error (RMSE)
- R² Score

---

## 📈 Visualizations

This project includes:

- Distribution of House Prices (Histogram)
- Correlation Heatmap
- Actual vs Predicted House Prices (Scatter Plot)

---

## 🔍 Key Findings

- House area showed the strongest influence on house prices.
- Bathrooms, parking spaces, and air conditioning also had a significant positive impact.
- Houses with better amenities generally had higher market values.
- The Linear Regression model achieved approximately **65% R² Score**, providing reasonably accurate predictions.
- The project demonstrates how machine learning can assist in estimating property prices.
